# Trigonometric Functions


## Q1


```python
from math import *

print("degrees 0 30 45 60 90")
import math
a=math.radians(0)
b=math.radians(30)
c=math.radians(45)
d=math.radians(60)
e=math.radians(90)

# write your code here
print("sin {} {} {} {} {}".format(math.sin(a),math.sin(b),math.sin(c),math.sin(d),math.sin(e)))
print("cos {} {} {} {} {}".format(math.cos(a),math.cos(b),math.cos(c),math.cos(d),math.cos(e)))
print("tan {} {} {} {} {}".format(math.tan(a),math.tan(b),math.tan(c),math.tan(d),math.tan(e)))
```

# Comment all previous code
